commandes description
git init créé un repository local
git clone copie la repository sur ton ordinateur(local)
git add ajoute un fichier dans le prochain commit
git commit sauvegarde les fichiers dans le dépôt avec un message
git push publie les changements sur le repository original
git branch affiche toutes les branches
git checkout change de branches
git merge fusionne la branche précisée sur la branche en cours
git log Montre tous les commits sur la branche en cours
git status affiche les fichiers en vert ajouté et en rouge ceux non ajoutés
git remote add [nom] [url] crée une connexion avec le repository d'origine
git pull télécharge les derniers changements sur le repository d'origin avec fusion
